# Phase 3 - AI Employee Setup

## Setup Objectives

Leave each participant with a useful AI employee that is grounded in company knowledge, shaped for a real role, and tested on actual tasks.

## Participant Setup Tracker

- Track cohort templates until the final roster is known
- Track workspace status, testing status, and handoff readiness

## Instructions Template

- Role description
- Company Brain context
- Tone and process rules
- Non-negotiables
- Boundaries

## Voice / Behavior Template

- Style defaults
- Review standards
- Escalation rules
- What the AI employee should never do

## Company Brain Loading Checklist

- Relevant docs selected
- Files loaded into the right workspace
- Context confirmed
- Missing knowledge flagged

## Testing Checklist

- Test task 1 completed
- Test task 2 completed
- Test task 3 completed
- Failure patterns reviewed

## Personal Setup Guide Template

- AI employee profile
- Instructions draft
- Setup checklist
- Test task log

## Handoff Checklist

- Owner confirmed
- Setup guide complete
- Open fixes logged
- Final notes delivered
