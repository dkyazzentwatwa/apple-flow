# Facilitator Command Center

## Delivery Checklist

- Confirm room and tech setup
- Confirm roster and cohort assignment
- Confirm slides, workbook, examples, and backup links
- Confirm recording owner and note capture owner

## Facilitation Notes

- Start every session with expected outcome and end state
- Tie examples to the client's real work whenever possible
- Keep role workflow examples separate from foundations
- Capture blockers as they appear for follow-up

## Timing Cues

- Review day-level pacing before each delivery block
- Leave time for artifact capture and QA handoff notes
- Flag any session that needs more facilitator or producer support
